# LineBankVerify
This project create for get data transaction in line chat send from bank and upload transaction to database

## 🛠️Develop on Python

## 🚀 Installation
```bash
# Clone git
git clone https://github.com/SilSea/LineBankVerify.git
cd LineBankVerify

```

## 📦Package Requirement
```bash
pip install pytest-playwright
playwright install
pip install pillow
```
